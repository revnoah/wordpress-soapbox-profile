WordPress plugin creates a soapbox for political parties.

== Description ==

WordPress plugin to create a soapbox platform for a political content relating to specific parties.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/soapbox-profile` directory, or install the plugin through 
the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Soapbox Profile screen to configure the plugin. You must choose either frontend or admin to see changes.
4. Customize output by adding any of the files below based on the front or admin settings you selected.
